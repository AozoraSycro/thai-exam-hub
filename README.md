# 📚 Thai Exam Hub (ไทยเอ็กแซมฮับ)

**Thai Exam Hub** คือเว็บแอปพลิเคชันคลังข้อสอบออนไลน์ฟรี สำหรับนักเรียนระดับชั้นมัธยมศึกษาตอนปลาย (ม.4-ม.6) ที่กำลังเตรียมตัวสอบเข้ามหาวิทยาลัยในระบบ **TCAS (ONET, TGAT, TPAT, A-Level)**

---

## ✨ จุดเด่นของโปรเจกต์
- **ฟรี 100%:** ไม่มีการเก็บค่าธรรมเนียม และไม่ต้องสมัครสมาชิก
- **ความเป็นส่วนตัวสูง:** ข้อมูลประวัติการทำข้อสอบถูกเก็บไว้ในเครื่องของคุณ (localStorage) ไม่มีการเก็บข้อมูลบนเซิร์ฟเวอร์
- **เฉลยละเอียด:** ทุกข้อสอบมาพร้อมคำอธิบายวิธีคิดที่เข้าใจง่าย
- **ใช้งานออฟไลน์ได้ (PWA):** สามารถติดตั้งแอปลงในมือถือและทำข้อสอบได้แม้ไม่มีอินเทอร์เน็ต
- **รองรับมือถือเต็มรูปแบบ:** ดีไซน์แบบ Mobile-first อ่านง่าย สบายตา

---

## 🚀 เริ่มต้นใช้งาน (Local Development)

1.  **Clone โปรเจกต์:**
    ```bash
    git clone https://github.com/your-username/thai-exam-hub.git
    ```
2.  **เปิดใช้งาน:**
    เนื่องจากมีการใช้ ES Modules และ Fetch API คุณจำเป็นต้องรันผ่าน Local Server:
    - หากใช้ **VS Code**: คลิกขวาที่ `index.html` แล้วเลือก **Open with Live Server**
    - หากใช้ **Python**: รันคำสั่ง `python -m http.server` ในโฟลเดอร์ `thai-exam-hub`

---

## 📱 วิธีการติดตั้งบนมือถือ (ใช้งานแบบแอปพลิเคชัน)

Thai Exam Hub รองรับระบบ PWA (Progressive Web App) ทำให้คุณสามารถติดตั้งลงบนหน้าจอมือถือเพื่อใช้งานแบบเต็มจอและรองรับการใช้งานออฟไลน์ได้:

**สำหรับผู้ใช้ iOS (iPhone/iPad):**
1. เปิดเว็บด้วยเบราว์เซอร์ **Safari**
2. แตะที่ปุ่ม **แชร์ (Share)** (ไอคอนสี่เหลี่ยมมีลูกศรชี้ขึ้น ด้านล่างจอ)
3. เลือกเมนู **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)**
4. กด **"เพิ่ม" (Add)** มุมขวาบน

**สำหรับผู้ใช้ Android:**
1. เปิดเว็บด้วยเบราว์เซอร์ **Chrome**
2. แตะที่เมนู **จุด 3 จุด** (มุมขวาบน)
3. เลือก **"ติดตั้งแอป" (Install App)** หรือ **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)**
4. กดยืนยันการติดตั้ง

## 🛠️ โครงสร้างไฟล์
- `index.html`: หน้าหลัก (Dashboard) และจุดเริ่มต้น
- `subject.html`: หน้าเลือกชุดข้อสอบรายวิชา
- `quiz.html`: ระบบทำข้อสอบและตรวจคะแนน
- `study.html`: ระบบคลังความรู้ (Study Hub)
- `data/`: คลังข้อสอบในรูปแบบ JSON
- `js/`: ตรรกะการทำงาน (Storage, Quiz Engine, App Logic)
- `css/`: สไตล์การแสดงผลที่รองรับมือถือและ AdSense
