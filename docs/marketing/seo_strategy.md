## 3. Recommended URL Structure
Clean, hierarchical URLs improve crawlability and CTR.

| Page Type | Recommended Pattern | Example |
|---|---|---|
| **Home** | `/` | `thaiexamhub.com/` |
| **Subject Hub** | `/subjects/{subject}/` | `/subjects/math/` |
| **Exam Year** | `/subjects/{subject}/{exam}-{year}/` | `/subjects/math/alevel-2567/` |
| **Mock Exam** | `/subjects/{subject}/mock-{id}/` | `/subjects/english/mock-tgat-1/` |

## 4. Zero-Budget "Growth Hacks"

### 1. Semantic Interlinking (The Web of Foxes)
- Every quiz page must link back to its **Subject Hub**.
- The **Subject Hub** must link to the **Top 5 most popular exams** in that category.
- Add a "Next Quiz" or "Recommended for You" section at the end of every quiz to reduce bounce rate.

### 2. Schema Markup (Structured Data)
Use JSON-LD to tell Google what we are:
- **FAQ Schema**: On subject pages to answer "What is TGAT 1?" or "When is A-Level 68?".
- **BreadcrumbList**: To show path in SERP (Home > Math > A-Level).
- **Quiz/EducationalOccupationalCredential**: Use structured data for quizzes if available.

### 3. Local Storage for "Freshness"
- Show a "User Streak" or "Recently Added" section on the home page. Google bots appreciate content that appears dynamic even if the site is static.

### 4. Image Optimization
- All diagrams MUST have `alt` tags in Thai containing keywords (e.g. `alt="เฉลยข้อสอบคณิตศาสตร์ A-Level เรื่องเซต"`).
- Use **WebP** format to ensure <2s load time on Thai 4G networks.

### 5. Social Sharing (Off-Page)
- Implement a "Share Score" button that generates a pre-filled message for X (Twitter) or LINE: 
  > "ฉันทำคะแนนได้ 80/100 ในแนวข้อสอบ TGAT 1 ที่ Thai Exam Hub! มาลองทำกัน: [URL]"
  - This generates natural social signals and potential backlinks from study blogs.
