## 3. URL Structure (Romanized Thai)

- **Subject Pages:** `/subjects/[subject-name]` (e.g., `/subjects/kanit-sat`)
- **Exam Pages:** `/exam/[exam-type]-[subject]-[year]` (e.g., `/exam/onet-kanit-2566`)
- **FAQ Pages:** `/faq/[question-slug]` (e.g., `/faq/onet-khue-arai`)
- **Results Page:** `/results` (dynamic)

## 7. Featured Snippet Targeting (10 H2 + Answers)

1. **H2: ONET สอบกี่วิชา?**
   - ตอบ: การสอบ ONET ระดับชั้น ม.6 สอบทั้งหมด 5 วิชาหลัก ได้แก่ 1. ภาษาไทย 2. คณิตศาสตร์ 3. ภาษาอังกฤษ 4. วิทยาศาสตร์ และ 5. สังคมศึกษา ศาสนา และวัฒนธรรม โดยใช้เวลาสอบวิชาละ 120 นาที

2. **H2: คะแนน ONET ประกาศวันไหน 2567?**
   - ตอบ: ตามปฏิทินของ สทศ. การประกาศผลสอบ ONET ปีการศึกษา 2566 (สอบปี 2567) จะประกาศในช่วงปลายเดือนมีนาคม ของปี พ.ศ. 2567 ผ่านทางเว็บไซต์ของ สทศ. (niets.or.th)

3. **H2: TGAT คืออะไร?**
   - ตอบ: TGAT (Thai General Aptitude Test) คือการวัดความถนัดทั่วไป ประกอบด้วย 3 ส่วน: การสื่อสารภาษาอังกฤษ, การคิดอย่างมีเหตุผล, และสมรรถนะการทำงานในอนาคต เป็นคะแนนสำคัญที่ใช้ในการคัดเลือกเข้ามหาวิทยาลัยระบบ TCAS

4. **H2: สอบ TCAS68 ต้องเตรียมตัวอย่างไร?**
   - ตอบ: 1. ศึกษาเกณฑ์การรับของคณะที่สนใจ 2. ตรวจสอบวิชาที่ต้องสอบ (TGAT/TPAT/A-Level) 3. วางแผนตารางอ่านหนังสือ 4. ฝึกทำข้อสอบเก่าสม่ำเสมอ และ 5. ติดตามวันรับสมัครให้แม่นยำ

5. **H2: TPAT3 สอบอะไรบ้าง?**
   - ตอบ: TPAT3 คือความถนัดทางวิทยาศาสตร์ เทคโนโลยี และวิศวกรรมศาสตร์ เน้นการทดสอบความคิดเชิงวิเคราะห์ ด้านตัวเลข ด้านมิติสัมพันธ์ และความรู้ทั่วไปด้านวิทยาศาสตร์และวิศวกรรม

6. **H2: ผลสอบ ONET ย้อนหลังดูที่ไหน?**
   - ตอบ: นักเรียนสามารถตรวจสอบผลสอบ ONET ย้อนหลังได้ที่เว็บไซต์ สทศ. (niets.or.th) โดยใช้เลขประจำตัวประชาชนและรหัสผ่านที่ลงทะเบียนไว้ หรือสอบถามได้ที่โรงเรียนที่เคยศึกษาอยู่

7. **H2: ภาษาไทย ม.6 ออกสอบเรื่องอะไรบ้าง?**
   - ตอบ: หัวข้อหลักที่ออกสอบบ่อย ได้แก่ การอ่านจับใจความ, การสะกดคำ, ราชาศัพท์, สำนวนไทย, และการใช้ภาษาให้ถูกต้องตามกาลเทศะ เน้นการคิดวิเคราะห์มากกว่าการท่องจำ

8. **H2: เตรียมสอบภาษาอังกฤษ ม.6 เริ่มตรงไหน?**
   - ตอบ: ควรเริ่มจากการเก็บคำศัพท์พื้นฐาน (Vocabulary) และทบทวนหลักไวยากรณ์ (Grammar) ที่พบบ่อย เช่น Tense, Passive Voice และ Relative Clause จากนั้นจึงเริ่มฝึกทำโจทย์ Reading

9. **H2: A-Level ต่างจาก 9 วิชาสามัญอย่างไร?**
   - ตอบ: A-Level คือชื่อเรียกใหม่ของการสอบ 9 วิชาสามัญเดิม โดยเน้นการวัดความรู้เชิงประยุกต์ (Applied Knowledge) มากขึ้น และมีการปรับโครงสร้างวิชาให้สอดคล้องกับหลักสูตรแกนกลางปัจจุบัน

10. **H2: สมัครสอบ TGAT TPAT ที่เว็บไหน?**
    - ตอบ: สมัครผ่านเว็บไซต์ mytcas.com ของ ทปอ. (ที่ประชุมอธิการบดีแห่งประเทศไทย) ซึ่งเป็นศูนย์กลางในการรับสมัครและจัดการสอบในระบบ TCAS ทั้งหมด

## 10. Google Search Console Setup Guide (ภาษาไทย)

1. **การยืนยันตัวตน:** ใช้ DNS TXT record หรืออัปโหลด HTML file ไปที่ GitHub repository (root)
2. **การส่ง Sitemap:** เข้าเมนู "Sitemaps" และระบุ `https://thaiexamhub.com/sitemap.xml`
3. **การตรวจสอบผล:** ตรวจสอบเมนู "Performance" รายสัปดาห์ เพื่อดูว่า Keyword ใดมี Impression สูงแต่ Click ต่ำ (เพื่อปรับ Meta Title)
4. **การทำ Index:** ใช้เมนู "URL Inspection" เพื่อส่งคำขอ Index หน้าเพจใหม่ทันทีที่เผยแพร่
5. **วิเคราะห์ Quick Wins:** ค้นหาคำค้นหาที่อยู่ในอันดับ 11-20 และเพิ่มเนื้อหาหรือ Internal Link เพื่อดันให้อยู่หน้าแรก
